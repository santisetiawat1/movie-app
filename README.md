# 🎬 My Movie App

A simple movie web application built using HTML, CSS, and JavaScript. This project focuses on displaying a static list of movies with a clean and responsive layout.

---

## 🌐 Live Demo
👉 https://movieapp-santi.netlify.app/

---

## 📌 About the Project
This project is a beginner-friendly frontend web project. It is designed as a simple movie display website without search functionality or external data integration.

Users can view a list of movies with basic information in a clean and structured layout.

---

## ✨ Features
- Static movie list display
- Clean and simple user interface
- Basic layout and styling practice

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript (basic)

---

## 🎯 Purpose
This project was created to practice frontend development fundamentals such as layout design, styling, and basic JavaScript structure.
